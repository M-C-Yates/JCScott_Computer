> An implementation of John C. Scotts 8-bit computer written in typescript.

## Specs

- `Memory`: 256 bytes
- 4x general purpose registers

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
